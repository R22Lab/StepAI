# AI Intensive Booking Website

A stylish, modern, minimalist black and white website for booking a 4-hour AI intensive.

## Features

1. **Favicon**: Added a robot emoji favicon for better branding
2. **Course Details**: Added a "What You'll Learn" section with course content
3. **Improved Form Validation**: Real-time validation with visual feedback
4. **Success Overlay**: Elegant modal confirmation instead of alert
5. **Accessibility**: Added ARIA attributes and proper labeling
6. **Date Restrictions**: Limited booking to 30 days in the future
7. **Loading State**: Visual feedback during form submission
8. **Formatted Display**: Dates and times formatted in a user-friendly way
9. **Enhanced Animations**: Smooth transitions and loading spinner
10. **Responsive Design**: Improved mobile experience

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## How to Use

Simply open `index.html` in any modern browser to view and use the booking form.

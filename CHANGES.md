# Улучшения дизайна сайта "Интенсив по ИИ" для Telegram Mini App

- Добавлено подключение Telegram Web Apps SDK для интеграции с Telegram

### 2. Обновление HTML структуры
- Добавлен viewport метатег с ограничением масштабирования для мобильных устройств

- Добавлены элементы для верхнего градиента и основного контента
- Обновлены ID и классы для лучшей совместимости с Telegram

- Создан мобильный дизайн с акцентом на UX в Telegram

- Обновлены стили формы под мобильные устройства
- Улучшена типографика и читаемость

- Обновлены стили оверлея для лучшего соответствия мобильному UX

### 4. Улучшение JavaScript для Telegram Web App
- Добавлена инициализация Telegram Web App
- Реализована интеграция с Main Button для отправки формы
- Добавлена обработка событий Telegram Web App
- Улучшена логика обработки формы с учетом особенностей Telegram
- Добавлена возможность закрытия приложения после успешного бронирования

### 5. Оптимизация 3D графики под мобильные устройства

### 6. Улучшение UX/UI для мобильных устройств
